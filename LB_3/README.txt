Калькулятор чайових

Мета

Зробити простий утилітарний веб-додаток для розрахунку чайових і поділу рахунку між людьми, з валідацією та збереженням налаштувань.

Функціонал

    Поля вводу:

        Сума рахунку (bill, десяткове число ≥ 0)

        Відсоток чайових (tipPercent, число від 0 до 50)

        Кількість людей (people, ціле ≥ 1)

    Кнопки швидкого вибору відсотка: 5%, 10%, 15%, 20%, Custom.

    Результати (автообчислення при зміні полів):

        Чайові загалом

        Загальна сума до сплати

        Сума на 1 людину (з округленням до 2 знаків)

    Кнопка Reset (скидає форму до дефолтів).

    Збереження останніх значень tipPercent і people у localStorage.

Валідація

    Порожні або нечислові значення → підсвітити поле, показати підказку «Введіть число».

    people < 1 → заблокувати обчислення, текст «Має бути ≥ 1».

    bill < 0 або tipPercent < 0 → помилка, не рахувати.

    Ввід з комою , → коректно парсити (замінити на крапку або дозволити обидва варіанти).

    Відображення 0.00 без артефактів (використовувати toFixed(2) або Intl.NumberFormat).
